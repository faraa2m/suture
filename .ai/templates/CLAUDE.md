# Suture Adapter

Use `.ai/` as the local orchestration layer.

## Startup

1. Read `.ai/router.md`.
2. Read `.ai/memory.md`.
3. Read `.ai/config.yaml`.
4. Apply `.ai/skills/caveman.md` for terse worker output when installed.
5. Apply `.ai/skills/ponytail.md` before adding code or dependencies when installed.

## Loop

- Route one bounded action at a time.
- Prefer Haiku-equivalent workers for mechanical work.
- Use Sonnet-equivalent synthesis for memory updates.
- Keep `.ai/memory.md` current.
