# Code of Conduct

This project is for practical, evidence-based work on local LLM orchestration.

## Expected Behavior

- Be direct, respectful, and technical.
- Assume contributors are here to improve the project.
- Critique ideas and changes, not people.
- Keep disagreement specific and actionable.
- Respect maintainers' scope decisions.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Demeaning language about identity, background, or experience level.
- Publishing private information without consent.
- Spam, trolling, or repeated off-topic arguments.
- Attempts to bypass project security or maintainer controls.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, and contributions that violate this code. Maintainers may also block accounts from participating in the project.

Report problems privately through GitHub security advisories when safety or abuse concerns require privacy.
